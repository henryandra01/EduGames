# EduGames
Jogos educativos interativos desenvolvidos para transformar aprendizado em diversão. A plataforma utiliza mecânicas de gamificação para estimular memória, raciocínio lógico, atenção, escrita e associação de conceitos através de desafios dinâmicos e intuitivos.Com atividades como Quiz com Pontuação e Níveis, Jogo da Memória Educativo, Caça-Palavras.
